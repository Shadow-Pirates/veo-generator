# 图片生成

## 功能说明

“图片生成”用于根据提示词生成图片，支持多页签、多张批量生成、下载与历史管理。

## 快速开始

1. 左侧进入“图片生成”
2. 点击“新建页签”
3. 输入提示词（尽量具体）：
   - 主体是什么
   - 场景/环境
   - 光线与氛围
   - 风格（写实/插画/电影感等）
4. 选择图片比例
5. 选择生成数量
6. 点击“生成”

## 提示词建议

- 多用“可观察”的描述：材质、光线、构图、镜头
- 少用“抽象词”当唯一信息：如“高级、舒服、震撼”

示例：

```text
一只橘猫坐在窗台上，雨天，玻璃有水滴，室内暖色灯光，浅景深，电影质感，超清细节
```

## 参考图

- 你可以上传参考图片来引导主体或构图
- 建议使用清晰、主体明确的图片

## 常见问题

### 为什么生成结果不稳定？

- 提示词过短或约束太少
- 参考图不明确或与提示词矛盾

### 怎么快速管理生成图片？

- 使用“历史记录”查看全部生成
- 对满意的结果加入“我的收藏”
