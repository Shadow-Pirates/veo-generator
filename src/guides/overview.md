# 开始使用

## 你能用交绘AI做什么

- 生成视频：输入故事板/提示词，提交后后台生成并可下载本地
- 生成图片：输入提示词，可批量生成并一键下载
- 任务管理：查看进行中任务、失败任务，支持继续轮询与状态更新
- 历史记录：统一管理生成结果，支持预览、下载、删除
- 模板管理：把常用提示词/参数保存为模板，提高复用效率
- 设置：配置数据目录、通知与存储清理

## 基本流程（推荐）

1. 在左侧底部点击“设置 API Key”并保存
2. 进入“视频生成”或“图片生成”，创建页签并填写内容
3. 点击“开始生成”，在右侧查看日志与进度
4. 生成完成后到“历史记录”集中管理与下载

## 数据存储位置

- 所有图片/视频/缩略图/数据库都存放在“设置 -> 数据目录”中
- 不建议把数据目录放在网盘同步目录或需要频繁权限弹窗的位置

## 常见问题

### 为什么启动后看起来像空白？

- 多数情况下是首次加载、正在读取本地数据或正在初始化任务列表
- 你可以先观察右侧“实时日志”和“任务管理”的任务状态
